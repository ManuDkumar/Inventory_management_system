\# Inventory Management System



\## Description

A Spring Boot backend application to manage products and inventory using REST APIs.



\## Tech Stack

\- Java

\- Spring Boot

\- PostgreSQL

\- Spring Data JPA



\## Features

\- Add product

\- Get all products

\- Get product by ID

\- Update product

\- Delete product



\## API Endpoints

\- POST /products

\- GET /products

\- GET /products/{id}

\- PUT /products/{id}

\- DELETE /products/{id}



\## Tools Used

\- Postman for API testing



\## How to Run

1\. Clone the repository

2\. Configure PostgreSQL in application.properties

3\. Run the Spring Boot application
